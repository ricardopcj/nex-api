# Fullstack Listing App (API + Web)

This project is a **simple fullstack application** composed of two repositories:

- **API** – built with Node.js and MongoDB (Mongoose)
- **Web** – built with React, consuming the API and displaying a list of items

The main goal of this project is to practice **API development, database modeling with MongoDB, and frontend consumption of REST APIs**.

---

## Project Architecture

This project is split into two separate repositories:

```text
📦 project
 ├── api/   → Node.js + MongoDB (Mongoose)
 └── web/   → React application
```

## 🚀 Tech Stack

### API

- Node.js
- JavaScript / TypeScript
- MongoDB
- Mongoose
- Express / Fastify
- dotenv

### Web

- React
- TypeScript
- Axios
- Vite

## Features

### API

- Connects to MongoDB using Mongoose
- Lists data from the database
- RESTful endpoints
- Environment-based configuration

### Web

- Consumes data from the API
- Displays a list of items
- Simple and clean UI
- Separation of concerns _(services, components)_
